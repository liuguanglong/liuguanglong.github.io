使用Blazor Server构建企业级应用(二) 使用MudBlazor搭建UI框架

本文地址：http://www.nintens.com/posts/blazorserver2/
作者邮箱：liuguanglong@yahoo.com
欢迎转载，请在明显位置给出出处及链接

  • MudBlazor基本配置
  • 配置UI Layout
  • 修改App应用图标，标题内容
  • 配置App菜单
  • 配置App LoginDisplay

一 引用MudBlazor

  1. 给blazor server添加MudBlazor包。在程序包管理器控制台窗口中输入以下命令
   Install-Package MudBlazor
  1. 将MudBlazor名字空间引入项目
    在blazor server项目下的_Imports.razor中添加以下代码
   @using MudBlazor
  1. 引入MudBlazor的字体和格式配置

打开_Layout.cshtml文件，在head部分中加入如下代码

    <link href="https://fonts.googleapis.com/css?family=Roboto:300,400,500,700&display=swap" rel="stylesheet" />
    <link href="_content/MudBlazor/MudBlazor.min.css" rel="stylesheet" />
    <script src="_content/MudBlazor/MudBlazor.min.js"></script>
  1. 将MudBlazor Service注入到App中

打开Startup.cs文件在ConfigureServices函数中添加以下代码。这部分代码包含了Snackbar的配置信息。 Snackbar的详细配置信息可以参考Mudblazor网站帮助信息。

public void ConfigureServices(IServiceCollection services)
{
    ... ...

    services.AddMudServices(config =>
    {
        config.SnackbarConfiguration.PositionClass = Defaults.Classes.Position.TopRight;

        config.SnackbarConfiguration.PreventDuplicates = false;
        config.SnackbarConfiguration.NewestOnTop = false;
        config.SnackbarConfiguration.ShowCloseIcon = true;
        config.SnackbarConfiguration.VisibleStateDuration = 10000;
        config.SnackbarConfiguration.HideTransitionDuration = 500;
        config.SnackbarConfiguration.ShowTransitionDuration = 500;
        config.SnackbarConfiguration.SnackbarVariant = Variant.Filled;
    });

    ... ...
}

二 搭建UI架构

  1. 配置UI布局

打开MainLayout.razor文件，删除所有内容。将以下代码复制到文件中

@inherits LayoutComponentBase

<MudThemeProvider/>
<MudDialogProvider 
    FullWidth="true"  
    MaxWidth="MaxWidth.Medium"    
    CloseButton="true"    
    DisableBackdropClick="true"   
    NoHeader="false"   
    Position="DialogPosition.TopCenter"/>
<MudSnackbarProvider />
<MudLayout>
    <MudAppBar Color="Color.Primary" Fixed="false" Dense="true" DisableGutters="true" Elevation="0">
        @if (_drawerOpen == true)
        {
            <MudElement Class="d-flex justify-center" Style="width:260px;"><img height="20" src="../logo_expand.png"></MudElement>
        }
        else
        {
            <MudElement Class="d-flex justify-center" Style="width:60px;"><img height="20" src="../logo_close.png"></MudElement>
        }

        <MudIconButton Icon="@Icons.Material.Filled.Menu" Color="Color.Inherit" Edge="Edge.Start" OnClick="@((e) => DrawerToggle())" />
        <MudText Align="Align.Left"><i>NinTens</i></MudText>
        <MudSpacer />
    </MudAppBar>
    <MudDrawerContainer Height="768px">
        <MudDrawer  Height="768px" Width="260px" @bind-Open="@_drawerOpen" Fixed="false" Variant="@DrawerVariant.Mini" OpenMiniOnHover="false" Elevation="0">
            <NavMenu />
        </MudDrawer>
        <MudMainContent Class="pt-16 px-16">
            <MudContainer MaxWidth="MaxWidth.ExtraLarge">
                @Body
            </MudContainer>
        </MudMainContent>
    </MudDrawerContainer>
</MudLayout>

@code {
    bool _drawerOpen = true;

    void DrawerToggle()
    {
        _drawerOpen = !_drawerOpen;
    }

    MudTheme MyCustomTheme = new MudTheme()
    {
        Palette = new Palette()
        {
            Primary = Colors.Blue.Darken2,
            Secondary = Colors.Red.Darken2,
            Tertiary = Colors.Green.Darken2
        }
    };
}

  1. 配置App应用的左侧菜单项目

打开NavMenu.razor文件，删除所有文件内容，复制已下代码

<MudNavMenu>
    <MudNavLink Href="/" Icon="@Icons.Filled.Home" IconColor="Color.Tertiary">Home</MudNavLink>
    <MudNavLink Icon="@Icons.Filled.Calculate" IconColor="Color.Tertiary" Href="counter">Counter</MudNavLink>
    <MudNavLink Icon="@Icons.Filled.WbSunny" IconColor="Color.Tertiary" Href="fetchdata"> Fetch data</MudNavLink>
</MudNavMenu>

将APP应用的图标放在wwwroot/目录下，打开_Layout.cshtml文件，在header部分代码中追加以下代码

    ... ...

    ... ...

  1. 修改APP应用标题

在Blazor Server应用中如果要修改App在浏览器中的标题，只需要在每个页面中设置PageTitle属性。
例如， 大概FetchData.razor文件，修改PageTitle属性内容为 Nintens - Weather forecast

<PageTitle>Nintens - Weather forecast</PageTitle>

运行程序效果如下图

  1. 在APP标题栏添加LoginDisplay

在Shared文件夹下添加razor组件 LoginDisplay.razor,复制以下代码

<MudText Align="Align.Right" Style="@($"color:{Colors.Yellow.Accent2};")">Hello, user@ninetens.com!</MudText>
<MudMenu Icon="@Icons.Material.Filled.Language" IconColor="@Color.Warning">
    @foreach (var culture in supportedCultures)
    {
        <MudMenuItem  Style="height:25px;width:140px;"><MudLink>@culture.DisplayName</MudLink></MudMenuItem>
    }
</MudMenu>

<MudMenu Style="width:120px;" Icon="@Icons.Material.Filled.Settings" IconColor="@Color.Warning" PositionAtCursor="true">
    <MudMenuItem Style="height:25px;width:100px;"><MudLink>Profile</MudLink></MudMenuItem>
    <MudMenuItem Style="height:25px;width:100px;"><MudLink>Logout</MudLink></MudMenuItem>
</MudMenu>
@code
{
    protected override void OnInitialized()
    {
        Culture = CultureInfo.CurrentCulture;
    }

    private CultureInfo Culture
    {
        get => CultureInfo.CurrentCulture;
        set
        {
        }
    }
    private CultureInfo[] supportedCultures = new[]
{
        new CultureInfo("en-US"),
        new CultureInfo("zh-CN"),
    };
