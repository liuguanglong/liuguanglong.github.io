使用Blazor Server构建企业级应用(一) -- 概述
2022-03-19 11:25 folder blazor   label blazor   asp.net core   .net core  

本文系列文章主要记述构建基于表单的简易Low-Code平台过程中的技术选型和实现的相关技术点。 整体架构采用Blazor Server, MudBlazor等技术实现。平台运行在CentOs 7服务器，基于Docker-Compose运行。 使用AzureDevops进行代码管理和发布。

本系列文章分别描述以下技术点

  1. Blazor Server项目创建
  2. EntityFrameWork以及MySql集成
  3. Docker-Compose制作和发布以及单机调试
  4. 客户端机密信息存储机制
  5. Docker自动化构建
  6. Linux平台生产机器发布
  7. Asp.net Identity身份验证集成
  8. Asp.net Identity画面定制化开发
  9. DDD基础框架
  10. 自动化测试
  11. 基于Cookie和Token的身份验证
  12. Blazor Server集成 WebAPI服务
  13. 使用MudBlazor
  14. 日志记录机制和日志查看功能
  15. Blazor Server的异步处理
  16. Blazor的多语言处理
  17. Blazor的业务数据的时区处理
